## 💡 实用技巧

### 多光标编辑
- 按住 `Option` 并点击多个位置创建多个光标
- `Cmd + D` 选择当前单词，继续按可以选择更多相同单词
- `Cmd + Shift + L` 选择所有匹配的单词

### 代码折叠
- `Cmd + Option + [` 折叠代码块
- `Cmd + Option + ]` 展开代码块
- `Cmd + K Cmd + 0` 折叠所有
- `Cmd + K Cmd + J` 展开所有

### 快速导航
- `Cmd + P` 然后输入文件名快速打开文件
- `Cmd + P` 然后输入 `@` 查看当前文件的符号
- `Cmd + P` 然后输入 `:` 跳转到指定行号

### 终端操作
- `Cmd + Shift + ` ` 新建终端
- `Cmd + \` ` 显示/隐藏终端
- `Ctrl + C` 终止当前命令
